```mermaid

    participant UI as 前端
    participant Routes as API路由

    participant Runner as AgentTaskRunner

    Flow->>Planner: update_plan(plan, step3)

    Note over Flow,ReAct: 阶段5：执行 Step 4（文件修复）

    Sandbox-->>ReAct: 修复成功
    ReAct-->>Flow: StepEvent(COMPLETED)

    ReAct-->>Flow: StepEvent(COMPLETED)
    Flow->>Planner: update_plan(plan, step7)

    AgentService->>DB: 保存所有事件

    Routes-->>UI: SSE 结束
    UI-->>User: 展示结果
```

## 3. PlannerReActFlow 状态机

```mermaid
stateDiagram-v2
    [*] --> IDLE
    IDLE --> PLANNING : 收到新消息

    PLANNING --> EXECUTING : plan 创建完成
    PLANNING --> COMPLETED : plan 为空/失败

    EXECUTING --> UPDATING : step 执行完成
    EXECUTING --> SUMMARIZING : 无剩余 step
    EXECUTING --> WAITING : 调用 message_ask_user
    EXECUTING --> COMPLETED : 任务超时/错误

    UPDATING --> EXECUTING : plan 更新完成
    UPDATING --> SUMMARIZING : 所有 step 已完成

    WAITING --> EXECUTING : 用户回复后继续
    WAITING --> PLANNING : 用户回复后重新规划

    SUMMARIZING --> COMPLETED : 汇总完成

    COMPLETED --> IDLE : 重置状态
    COMPLETED --> [*] : 任务结束
```

## 4. 单 Step 执行详细流程

```mermaid
flowchart TD
    Start([ReActAgent.execute_step]) --> BuildContext[构建 plan_context]
    BuildContext --> FormatPrompt[格式化 EXECUTION_PROMPT]
    FormatPrompt --> Invoke[调用 BaseAgent.invoke]
    Invoke --> LLM1[LLM 第1次调用]
    LLM1 --> HasToolCall{是否有 tool_calls?}

    HasToolCall -->|否| ParseResult[解析 JSON 结果]
    ParseResult --> UpdateStep[更新 step 状态为 COMPLETED]
    UpdateStep --> YieldStep[Yield StepEvent]
    YieldStep --> End([返回])

    HasToolCall -->|是| LimitCheck{超过 max_iterations_per_step?}
    LimitCheck -->|是| YieldError[Yield ErrorEvent]
    YieldError --> End

    LimitCheck -->|否| ExecuteTool[执行工具]
    ExecuteTool --> YieldCalling[Yield ToolEvent CALLING]
    ExecuteTool --> GetResult[获取 ToolResult]
    GetResult --> YieldCalled[Yield ToolEvent CALLED]

    YieldCalled --> NeedScreenshot{是 browser 工具?}
    NeedScreenshot -->|是| UploadScreenshot[上传截图到 OSS]
    UploadScreenshot --> AttachURL[附加 screenshot URL]
    NeedScreenshot -->|否| ReflectCheck{到达 reflection_interval?}
    AttachURL --> ReflectCheck

    ReflectCheck -->|是| InjectReflection[插入反思 user message]
    ReflectCheck -->|否| LLMNext[调用 LLM 下一次]
    InjectReflection --> LLMNext
    LLMNext --> LLM1
```

## 7. 防跑偏机制流程

```mermaid
flowchart TD
    Start([用户请求]) --> CreatePlan[Planner 创建 Plan]
    CreatePlan --> HasGoal{Plan 包含 goal 和 steps?}
    HasGoal -->|否| Error[返回错误]
    HasGoal -->|是| ExecuteStep[取出第一个 step]

    ExecuteStep --> InjectContext[注入全局计划上下文]
    InjectContext --> ReActLoop[ReAct 工具循环]

    ReActLoop --> SingleTool[单次只调一个工具]
    SingleTool --> CheckBudget{超过 max_iterations_per_step?}
    CheckBudget -->|是| StepFail[Step 失败]
    CheckBudget -->|否| RunTool[执行工具]

    RunTool --> Reflect{到达 reflection_interval?}
    Reflect -->|是| ReflectPrompt[插入反思提示]
    Reflect -->|否| NextLLM[下一次 LLM 调用]
    ReflectPrompt --> NextLLM

    NextLLM --> MoreTool{还需要工具?}
    MoreTool -->|是| SingleTool
    MoreTool -->|否| StepDone[Step 完成]

    StepDone --> EarlyCheck{enable_early_completion 且检测到提前完成?}
    EarlyCheck -->|是| CheckGoal[检查全局目标是否满足]
    CheckGoal -->|是| SkipRemaining[跳过剩余 steps]
    CheckGoal -->|否| UpdatePlan
    EarlyCheck -->|否| UpdatePlan[Planner 更新 Plan]

    SkipRemaining --> Summarize[进入 SUMMARIZING]
    UpdatePlan --> HasMoreStep{还有 step?}
    HasMoreStep -->|是| ExecuteStep
    HasMoreStep -->|否| Summarize

    Summarize --> FinalLLM[LLM 生成最终回复]
    FinalLLM --> Done([DoneEvent])

    StepFail --> UpdatePlan
```

```mermaid

    Flow --> PlanEvent

    ReAct --> ToolEvent
    Flow --> WaitEvent
    Flow --> DoneEvent
    Flow --> ErrorEvent

    PlanEvent --> Persistence[持久化 + SSE 推送]
    StepEvent --> Persistence
    ToolEvent --> Persistence
    MessageEvent --> Persistence
    TitleEvent --> Persistence
    WaitEvent --> Persistence

    DoneEvent --> Persistence

    Persistence --> DB[(PostgreSQL)]
    Persistence --> SSE[SSE 推送]
    SSE --> UI[前端展示]
```

### 事件类型说明

| 事件类型 | 触发方 | 典型状态/角色 | 说明 |
|---|---|---|---|
| `plan` | PlannerAgent / PlannerReActFlow | `created` / `updated` / `completed` | 计划的创建、更新与完成 |
| `title` | AgentTaskRunner | - | 会话标题 |
| `step` | ReActAgent / PlannerReActFlow | `started` / `completed` / `failed` | 步骤开始、完成、失败 |
| `message` | AgentService / ReActAgent / summarize | `user` / `assistant` | 用户消息与助手消息 |
| `tool` | ReActAgent / BaseAgent | `calling` / `called` | 工具调用前后 |
| `wait` | PlannerReActFlow | - | 等待用户回复 |
| `error` | PlannerReActFlow / AgentService | - | 步骤失败或服务异常 |
| `done` | PlannerReActFlow | - | 任务正常结束 |
